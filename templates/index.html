About This Project

This Vehicle Parking System was developed as part of the MAD I project in the IIT B.S Data Science program.

The system provides comprehensive parking management capabilities including lot creation, spot reservation, user management, and reporting features.

Built with Flask, Bootstrap 5, and SQLite, this application demonstrates modern web development practices while solving real-world parking challenges.
